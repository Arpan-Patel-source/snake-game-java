# 🐍 Snake Game (Java)

## Overview
A classic Snake Game built using Java Swing with keyboard controls and real-time rendering.

## Features
- Smooth snake movement
- Food generation at random positions
- Score tracking
- Collision detection (walls + self)

## Tech Stack
- Java
- Swing (GUI)

## How to Run
1. Download the project
2. Compile:
   javac SnakeGame.java
3. Run:
   java SnakeGame

## Controls
Arrow Keys → Move snake
